# Предметы и инвентарь

В TaleCore каждый значимый предмет имеет **карточку** — краткое описание, эффект и состояние. Это убирает споры о «а что именно делает этот меч?» и делает предметы осязаемыми.

---

## Карточка предмета

| Поле | Содержание |
|---|---|
| **Название** | Название предмета |
| **Тип** | Оружие / Снаряжение / Расходник / Магический / Особый |
| **Описание** | Внешний вид, материал, история (если есть) |
| **Эффект** | Конкретный игровой эффект (бонусный кубик, состояние, особое действие) |
| **Дистанция** | Для оружия: ближняя / средняя / дальняя |
| **Состояние** | Исправен / Изношен / Повреждён / Сломан |

---

## Использование предметов

**Вне боя:** использование предмета не требует отдельного действия — описывай и применяй.

**В бою:** использование предмета стоит **одно действие в ход** («Использовать предмет»).

---

## Состояние предметов

| Состояние | Описание | Эффект |
|---|---|---|
| **Исправен** | Предмет в рабочем состоянии | Нет штрафов |
| **Изношен** | Видны признаки долгого использования | Мастер может потребовать проверку при критической нагрузке |
| **Повреждён** | Явные дефекты, работает частично | -1 кубик к броскам, связанным с предметом |
| **Сломан** | Предмет не функционирует | Использование невозможно до ремонта |

---

## Оружие

Оружие — особый тип предмета. Его карточка всегда включает дистанцию и особые эффекты (если есть).

Примеры:
- **Кинжал** — Ближняя дистанция. Можно скрыть под одеждой (проверка Действия, сложность 1).
- **Длинный лук** — Дальняя дистанция. Требует свободного пространства, неэффективен в тесных помещениях.
- **Алебарда** — Средняя и Ближняя дистанция. Дополнительный кубик при атаке с высоты.

---

## Расходники

Предметы с ограниченным числом использований (зелья, свитки, боеприпасы).

- Зелья применяются действием «Использовать предмет»
- Свитки: содержат одно заклинание, сгорают при использовании
- Боеприпасы: мастер отслеживает приблизительно (много / мало / нет)

---

## Магические предметы

Магические предметы имеют дополнительный раздел в карточке — описание магического эффекта. Некоторые могут требовать проверки Заклинания для активации.

---

## Нагрузка и вес

TaleCore не использует числовые показатели веса. Нагрузка — **нарративная**: мастер описывает ситуации, когда персонаж явно перегружен, и может добавить осложнение или потребовать проверки Самочувствия.

---

## Создание предметов

Персонаж может попытаться создать предмет из доступных материалов — проверка релевантной характеристики (Мышление для алхимии, Действие для ловушек). Сложность определяет мастер по сложности задачи.
