## Характеристики

Каждая характеристика имеет значение от **1 до 4** — это количество d6, бросаемых при проверке.

> **Результаты броска:** 6 = 2 успеха | 4–5 = 1 успех | 1–3 = 0 успехов

| Характеристика | Значение (1–4) | Назначение |
|---|---|---|
| **Внимание** | | Восприятие, наблюдательность, реакция |
| **Действие** | | Физические действия, ловкость, скрытность |
| **Заклинание** | | Магические способности, контроль маны |
| **Мышление** | | Интеллект, знания, анализ, хитрость |
| **Общение** | | Убеждение, харизма, обман, запугивание |
| **Самочувствие** | | Выносливость, стойкость к боли, здоровье |
| **Сражение** | | Боевые навыки, атака, защита |

## Биография

> Каждый блок должен содержать **минимум один абзац** текста. Биография — основа для формирования перков и умений.

### Желания
*Чего персонаж хочет достичь? Его долгосрочные стремления и мечты.*

---

### Мотивы
*Почему персонаж делает то, что делает? Что заставляет его двигаться вперёд?*

---

### Образ
*Как выглядит персонаж? Внешние вид, отличительные черты или поведенческие особенности.*

---

### Предыстория
*Откуда персонаж? Какие ключевые события его прошлого сформировали его?*

---

### Проблемы
*С какими трудностями он сталкивается? Незакрытые долги, враги, проклятия, страхи.*

---

### Слабости
*Что может использовать противник? Моральные, физические или психологические уязвимости.*

---

### Черты характера
*Как персонаж ведёт себя? Привычки, манеры, реакции в стрессовых ситуациях.*

---
