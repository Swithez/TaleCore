# Персонажи кампании

В этой папке хранятся листы персонажей игроков.

Каждый персонаж находится в отдельной подпапке:

```
Персонажи/
├── ИмяПерсонажа1/
│   ├── лист.md           ← основной лист персонажа
│   ├── биография.md      ← развёрнутая биография (опционально)
│   ├── инвентарь.md      ← вещи и снаряжение (опционально)
│   └── заметки.md        ← личные заметки игрока (опционально)
│
└── ИмяПерсонажа2/
    └── лист.md
```

---

## Права доступа

- **Игрок** видит только свою папку персонажа
- Чужие персонажи видны **только** если игрок включил
  параметр «Виден для других игроков» в своём профиле
- **Мастер** видит все папки персонажей в кампании

---

## Шаблон листа персонажа

При создании персонажа система автоматически создаёт файл `лист.md`.
Структуру листа смотри в: `Персонаж/Лист персонажа.md`
