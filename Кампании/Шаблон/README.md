# Структура кампании

Это шаблонная структура папок для новой кампании по TaleCore.
Скопируй эту папку целиком, переименуй по названию кампании и начни заполнять.

---

## Структура папок

```
НазваниеКампании/
│
├── Бестиарий/              ← враги, монстры, боевые НПС (со статблоком)
│   └── Шаблон НПС.md
│
├── НПС/                    ← именные персонажи мира (союзники, торговцы, информаторы)
│   └── Шаблон НПС.md
│
├── Локации/                ← места: города, здания, подземелья, регионы
│   └── Шаблон локации.md
│
├── Фракции/                ← гильдии, ордены, организации, политические группы
│   └── Шаблон фракции.md
│
├── Квесты/                 ← активные и завершённые сюжетные нити
│   └── Шаблон квеста.md
│
├── События/                ← хроника ключевых событий, изменивших мир кампании
│   └── Шаблон события.md
│
├── Сессии/                 ← журнал сессий: что было, что решено, что готовить
│   └── Шаблон сессии.md
│
├── Предметы/               ← артефакты, улики, значимые предметы с историей
│   └── Шаблон предмета.md
│
├── Перки/                  ← кастомные перки для этой кампании
│   └── Шаблон перка.md
│
├── Рассы/                  ← кастомные или изменённые расы (если отличаются от стандарта)
│   └── Шаблон рассы.md
│
├── Умения/                 ← кастомные умения для этой кампании
│   └── Шаблон умения.md
│
└── Персонажи/              ← листы персонажей игроков (создаются автоматически)
    └── ИмяПерсонажа/
        └── лист.md
```

## Бестиарий vs НПС — в чём разница?

| | Бестиарий | НПС |
|---|---|---|
| **Для кого** | Враги, монстры, боевые существа | Союзники, торговцы, информаторы |
| **Акцент** | Статблок, тактика, угрозность | Отношения, информация, история |
| **HP и характеристики** | Обязательно | Опционально |
| **История взаимодействий** | Нет | Да |

## Как создать кампанию

1. Открой личный кабинет и нажми «Создать кампанию» — дай ей название
2. Система создаст папку с базовой структурой автоматически
3. Заполни мировую концепцию — см. `Кампании/Инструкция по созданию.md`
4. Пригласи игроков: они выбирают кампанию при регистрации и создают персонажей

---

## С чего начать наполнение

**Минимум для первой сессии:**
- 1–2 локации (`Локации/`)
- 2–3 НПС с которыми группа встретится (`НПС/`)
- 1 активный квест (`Квесты/`)
- Заполни журнал после сессии (`Сессии/`)

**По ходу кампании добавляй:**
- Фракции по мере их появления в игре
- События после каждого ключевого момента
- Предметы при появлении значимых артефактов

---

## Папка «Персонажи»

Внутри каждой кампании есть папка `Персонажи/`, где хранятся листы игровых персонажей.
Каждый игрок видит **свою** папку. Чужие персонажи видны только если игрок включил
параметр **«Виден для других игроков»** в своём профиле.
Мастер видит все папки персонажей в кампании.
