# Структура кампании

Это шаблонная структура папок для новой кампании по TaleCore.
Скопируй эту папку целиком, переименуй по названию кампании и начни заполнять.

---

## Структура папок

```
НазваниеКампании/
├── README.md                  ← этот файл (удали или перепиши для игроков)
│
├── Бестиарий/                 ← НПС и враги
│   └── Шаблон НПС.md
│
├── Локации/                   ← Места, которые посещают игроки
│   └── Шаблон локации.md
│
├── Перки/                     ← Перки, доступные в этой кампании
│   └── Шаблон перка.md
│
├── Рассы/                     ← Расы, если есть кастомные или ограничения
│   └── Шаблон рассы.md
│
├── Умения/                    ← Умения, доступные в этой кампании
│   └── Шаблон умения.md
│
└── Персонажи/                 ← Папки с листами персонажей игроков
    └── ИмяПерсонажа/
        └── лист.md
```

---

## Как создать кампанию

1. Дай кампании **название** в личном кабинете (раздел «Создать кампанию»)
2. Система создаст папку кампании с базовой структурой автоматически
3. Заполни мировую концепцию — см. `Кампании/Инструкция по созданию.md`
4. Пригласи игроков: они выбирают кампанию при регистрации и создают персонажей

---

## Папка «Персонажи»

Внутри каждой кампании есть папка `Персонажи/`, где хранятся листы игровых персонажей.
Каждый игрок видит **свою** папку. Чужие персонажи видны только если игрок включил
параметр **«Виден для других игроков»** в своём профиле.

---

## Советы мастеру

- Создавай отдельный файл на каждого НПС в папке `Бестиарий/`
- Веди журнал сессий в отдельной папке `Сессии/` (создай вручную)
- Храни карты и описания мира в `Локации/`
- Используй AI-ассистента TaleForge для генерации и редактирования контента
