# TaleCore

**TaleCore** — движок для настольных ролевых игр, ориентированный на продвинутых мастеров и сторителлинг. Главная цель — убрать лишние броски, неиспользуемые механики и билдостроение, оставив место для живой истории, характеров и решений.

---

## Основные принципы

- **История прежде всего.** Механики служат нарративу, а не наоборот.
- **Персонаж — это биография, а не набор цифр.** Умения и перки вытекают из прошлого персонажа.
- **Мастер — соавтор.** Он формирует стартовый набор умений и перков, корректирует его в ходе игры.
- **Минимум лишних бросков.** Кубики бросаются только тогда, когда исход действительно неочевиден.

---

## Чем TaleCore отличается от D&D

| Аспект | D&D | TaleCore |
|---|---|---|
| Создание персонажа | Выбор класса, расы, распределение очков | Написание биографии, мастер выдаёт перки/умения |
| Прогрессия | Уровни, фиксированный левел-ап | Изменение набора перков/умений по ходу истории |
| Расы | Бонусы к характеристикам | Колорит, уникальный игровой опыт |
| Дистанция | Футовая сетка | Абстрактно: ближний / средний / дальний |
| Классы | Фиксированные | Отсутствуют — у каждого уникальный набор |
| Вдохновение | Однократный бонус | **Тейлы** — валюта с несколькими применениями |
| Магия | Слоты заклинаний | Знаки (свободно) и Ритуалы (за Тейл) |

## Быстрый старт

### Для игрока
1. Прочитай раздел [Лист персонажа](Персонаж/Лист%20персонажа.md).
2. Напиши биографию по шести блокам: Желания, Мотивы, Предыстория, Проблемы, Слабости, Черты характера.
3. Передай биографию мастеру — он сформирует стартовый набор перков и умений.
4. Распредели значения 7 характеристик (мастер укажет доступные очки).

### Для мастера
1. Прочитай [Инструкцию по созданию кампании](Кампании/Инструкция%20по%20созданию.md).
2. Изучи [Механики](Механики/) — особенно Проверку характеристик, Тейлы и Бой.
3. На основе биографий игроков сформируй стартовые перки и умения для каждого персонажа.
4. Используй шаблоны из папки `Кампании/Шаблон/` для создания НПС, локаций и предметов.
